Part 2: Simulation via numerical integration

  • The cloth seems to take more time to reach the balance state (non-moving state) with low ks compared to high ks. Also, the cloth looks to be more "saggy".
  • When we decrease density, we effectively decrease mass, so the cloth doesn't sag as much. Additionally, since the mass is decreased, the cloth takes more time to reach equilibrium because it has less inertia.
  • If damping is decreased to 0, the cloth will fluctuate for a long time since there is little energy lost. On the contrary, if damping is increased, the cloth looks to be in some kind of thick liquid.
